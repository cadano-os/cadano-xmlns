Cadano UI XML Namespace

This namespace is part of the Cadano Operating System's user interface definition. It is used to identify and style UI components in a structured XML format.

Defined Attributes

  • cadano:id – Unique identifier of the element
  • cadano:ref – References another element
  • cadano:layout – Layout type (e.g., "vertical", "grid")
  • cadano:order – Order index in layout
  • cadano:slot – Named slot for content projection
  • cadano:style – UI style class
  • cadano:theme – Theme name ("light", "dark", etc.)
  • cadano:color – Text color
  • cadano:bgcolor – Background color
  • cadano:opacity – Transparency level (0.0 to 1.0)
  • cadano:border – Border definition
  • cadano:radius – Border radius
  • cadano:margin – Outer spacing (e.g., "10px")
  • cadano:padding – Inner spacing
  • cadano:onclick – Triggered when clicked
  • cadano:onhover – Triggered on hover
  • cadano:enabled – Whether the element is active
  • cadano:visible – Visibility toggle
  • cadano:draggable – Enable drag functionality
  • cadano:lang – Language key (for translation)
  • cadano:text – Element inner text
  • cadano:tooltip – Tooltip on hover
  • cadano:placeholder – Placeholder text
  • cadano:bind – Bind to variable or setting
  • cadano:model – Two-way binding
  • cadano:if – Conditional rendering
  • cadano:state – Initial state of the component
  • cadano:watch – Execute on value change
  • cadano:animate – Animation type ("fade-in", "bounce")
  • cadano:delay – Delay before animation
  • cadano:duration – Animation duration
  • cadano:ease – Easing style ("ease-in", "linear")
  • cadano:transition – Full CSS transition string
  • cadano:load-src – Load content from external source
  • cadano:component – Embed another component
  • cadano:import – Import external UI file
  • cadano:role – ARIA role for accessibility
  • cadano:debug – Enable developer debug mode
  • cadano:snap – Snap to parent/container edge

Example Usage

<view xmlns:cadano="https://cadano-os.github.io/cadano-xmlns/ns/ui">
    <button cadano:id="btnOK" cadano:style="primary">OK</button>
</view>

For more information, visit the Cadano XMLNS GitHub Repository.
